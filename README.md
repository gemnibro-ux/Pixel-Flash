# Assembly Pixel

PF (Полн. Pixel Flash) — быстрый и лёгкий язык программирование. На нём можно писать код изменять регистры, очищать. Этот язык программирование подходит для энтузиастов, исследователям либо кем кто хочет просто посмотреть как работает код. Язык похож на ассемблер, но облегчен!

## Возможности

- Легендарный и простой дизайн
- Очистка вывода 
- Показ ошибок в выводе
- Безопасность! Он не влияет на ядро

## Функции

1. PASTE РЕГИСТР ЗНАЧЕНИЕ:
- На нём можно изменить значение регистра
- Так же можно изменить значение через другие переменные

2. ADD РЕГИСТР1 РЕГИСТР2:
- Это то же самое, но там прибавлять

3. REG РЕГИСТР
- Эта функция печатает значение определённого регистра
- Есть фишка: REG __REGS. Он печатает все регистры

4. MSG СТРОКА
- На нём можно писать слова. Пример: MSG: Hi, bro!

5. MAKE РЕГИСТР ЗНАЧЕНИЕ
- Создаёт регистры собственного значение

6. DEL РЕГИСТР
- Эта функция удаляет определённый регистр

7. MOVE РЕГИСТР1 РЕГИСТР2
- Функция, которая перемешивает второй регистр с первым

8. REMOVE РЕГИСТР
-  Функция, которая возращает старое значение определённого регистра изменившиеся через MOVE

9. HALT
- Эта функция просто останавливает весь код

10. CLEAR РЕГИСТР
- Она очищает определённый регистр до полного нуля
- Фишка: CLEAR __REGS. Очищает все регистры до полного нуля.
